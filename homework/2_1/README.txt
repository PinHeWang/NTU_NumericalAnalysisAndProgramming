Monkeys love to have more food in the morning, rather than in the afternoon or evening. If A, B, and C are the numbers of bananas given for the breakfast, lunch, and dinner, respectively. The monkeys will be happy if A is larger than or equal to B, and B is larger than or equal to C. Otherwise they will be angry. Note A, B, and C are all positive integers.

Implement the given function as following:
return True if the monkeys are happy; return False if the monkeys are angry.
