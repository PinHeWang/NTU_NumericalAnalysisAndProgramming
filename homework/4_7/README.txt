Write a function which returns a list of strings. The list should contain the standard 52-card deck, that is:

['CLUB-A', 'CLUB-2', 'CLUB-3',...,'CLUB-J', 'CLUB-Q', 'CLUB-K',
 'DIAMOND-A', 'DIAMOND-2',...,'DIAMOND-K', 
 'HEART-A', 'HEART-2',...,'HEART-K',
 'SPADE-A', 'SPADE-2',...,'SPADE-K',]
Note: all of the strings should fellow the format above: all captital letters for the suits (CLUB/DIAMOND/HEART/SPADE), followed with a hyphen (-), and the kind (A, 2, 3,..., 9, 10, J, Q, K).
