Implement the function below which takes a single integer input, finds out all of the factors, and returns the sum of them. 
For example,

if the input is 4, which has 1, 2, 4 as its factors. Your code should return 1+2+4, which is 7.
if the input is 10, which has 1, 2, 5, 10 as its factors. Your code should return 1+2+5+10, which is 18.
